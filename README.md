# C-Programs
This repository contains my beginner projects in the C programming language.

## Programs in this repo
- **alphabetwhileloop.c** → Prints A to Z using a while loop
- **calculator.c** → Simple calculator program
- **evenodd.c** → Checks if a number is even or odd

## Getting Started
To run these programs:
1. Download the `.c` file.
2. Compile using `gcc filename.c -o output`.
3. Run using `./output`.
